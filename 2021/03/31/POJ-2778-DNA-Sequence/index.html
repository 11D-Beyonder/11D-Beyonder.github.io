POJ 2778 DNA Sequence

发表于2021-03-31|更新于2023-01-31|题解
|字数总计:1.4k|阅读时长:7分钟|阅读量:

Description

It’s well known that DNA Sequence is a sequence only contains A\text AA, C\text CC, T\text TT and G\text GG, and it’s very useful to analyze a segment of DNA Sequence. For example, if a animal’s DNA sequence contains segment ATC\text{ATC}ATC then it may mean that the animal may have a genetic disease. Until now scientists have found several those segments, the problem is how many kinds of DNA sequences of a species don’t contain those segments.
Suppose that DNA sequences of a species is a sequence that consist of A\text AA, C\text CC, T\text TT and G\text GG, and the length of sequences is a given integer nnn.

Input

First line contains two integer mmm (0≤m≤100 \le m \le 100≤m≤10), nnn (1≤n≤20000000001 \le n \le 20000000001≤n≤2000000000). Here, mmm is the number of genetic disease segment, and n is the length of sequences.
Next mmm lines each line contain a DNA genetic disease segment, and length of these segments is not larger than 101010.

Output

An integer, the number of DNA sequences,  mod 100000\bmod 100000mod100000.

Idea

设字典树可接受字符集为小写英文字母。将 mmm 个病毒串插入字典树，并为病毒串末尾打上标记。我们的目标是，从树根出发，走 nnn 层构造出一个字符串 sss，且 sss 的任意子串都不是病毒串。
上述方法存在两个问题。第一，我们仅仅在字典树上插入了病毒串，若要使得构造的字符串 sss 能够使用字符集中的所有字母，需要存一棵 262626 叉树，这显然是不可能的。第二，如果我们仅仅在病毒串的结尾打标记，会对病毒串的检索遗漏，比如插入两个字符串 ACG\text{ACG}ACG 和 C\text CC，我们在两个字符串的结尾 G\text GG 和 C\text{C}C 打上标记；然而，若游走到了树上表示 AC\text{AC}AC 的位置，实际上已经包含病毒串 C\text CC，但是我们并没有在这个位置打上标记。
