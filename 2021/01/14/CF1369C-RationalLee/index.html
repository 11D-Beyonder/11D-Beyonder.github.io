CF1369C - From Codeforces Round #652 (Div. 2)

Lee just became Master in Codeforces, and so, he went out to buy some gifts for his friends. He bought n integers, now it’s time to distribute them between his friends rationally…
Lee has nnn integers a1,a2,…,ana_1,a_2,…,a_na1​,a2​,…,an​ in his backpack and he has kkk friends. Lee would like to distribute all integers in his backpack between his friends, such that the iii-th friend will get exactly wiw_iwi​ integers and each integer will be handed over to exactly one friend.
Let’s define the happiness of a friend as the sum of the maximum and the minimum integer he’ll get.
Lee would like to make his friends as happy as possible, in other words, he’d like to maximize the sum of friends’ happiness. Now he asks you to calculate the maximum sum of friends’ happiness.

Input

The first line contains one integer ttt (1≤t≤104)(1≤t≤10^4)(1≤t≤104) — the number of test cases.
Next 3t3t3t lines contain test cases — one per three lines.
The first line of each test case contains two integers nnn and kkk (1≤n≤2⋅1051≤n≤2⋅10^51≤n≤2⋅105; 1≤k≤n1≤k≤n1≤k≤n) — the number of integers Lee has and the number of Lee’s friends.
The second line of each test case contains nnn integers a1,a2,…,ana_1,a_2,…,a_na1​,a2​,…,an​ (−109≤ai≤109−10^9≤a_i≤10^9−109≤ai​≤109) — the integers Lee has.
The third line contains kkk integers w1,w2,…,wkw_1,w_2,…,w_kw1​,w2​,…,wk​ (1≤wi≤n1≤w_i≤n1≤wi​≤n; w1+w2+…+wk=nw_1+w_2+…+w_k=nw1​+w2​+…+wk​=n) — the number of integers Lee wants to give to each friend.
It’s guaranteed that the sum of n over test cases is less than or equal to 2⋅1052⋅10^52⋅105.

Output

For each test case, print a single integer — the maximum sum of happiness Lee can achieve.

Note

In the first test case, Lee should give the greatest integer to the first friend (his happiness will be 17+1717+1717+17) and remaining integers to the second friend (his happiness will be 13+113+113+1).
In the second test case, Lee should give $\{10,10,11\}$ to the first friend and to the second friend, so the total happiness will be equal to (11+10)+(11+10)(11+10)+(11+10)(11+10)+(11+10).
In the third test case, Lee has four friends and four integers, it doesn’t matter how he distributes the integers between his friends.

Translation

有 nnn 个整数 a1,a2,⋯ ,ana_1,a_2,\cdots,a_na1​,a2​,⋯,an​，Lee 想要将它们放进 kkk 个有标号的盒子，使得标号为 iii 的盒子中恰有 wiw_iwi​ 个数。
定义一个盒子的价值为其中最大数与最小数之和，定义一种放法的总价值为所有盒子的价值之和。Lee 希望求出总价值的最大值。

Idea

显然要让 kkk 个最大的 aia_iai​ 发挥作用，让 kkk 个最小的 aia_iai​ 尽可能少量的产生贡献。

对于 wi=1w_i=1wi​=1 的袋子，将最大的 aia_iai​ 放入，产生的贡献为 2ai2a_i2ai​，较大值直接产生两倍贡献，这肯定是最优的；对于 wi>1w_i>1wi​>1 的袋子，将最大的 ai(max⁡)a_{i(\max)}ai(max)​ 放入，然后将最小的 ai(min⁡)a_{i(\min)}ai(min)​ 放入，中间依次填充较小的 aia_iai​，这样就能尽可能屏蔽 kkk 个最小的 aia_iai​ 产生的贡献。
