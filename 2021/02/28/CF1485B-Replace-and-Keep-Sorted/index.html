CF1485B Replace and Keep Sorted

发表于2021-02-28|更新于2023-01-31|题解
|字数总计:768|阅读时长:4分钟|阅读量:

Given a positive integer kkk, two arrays are called kkk-similar if:

  • they are strictly increasing;
  • they have the same length;
  • all their elements are positive integers between 111 and kkk (inclusive);
  • they differ in exactly one position.

You are given an integer kkk, a strictly increasing array aaa and qqq queries. For each query, you are given two integers li≤ril_i≤r_ili​≤ri​. Your task is to find how many arrays bbb exist, such that bbb is kkk-similar to array [ali,ali+1…,ari][a_{l_i},a_{l_i+1}…,a_{r_i}][ali​​,ali​+1​…,ari​​].

Input

The first line contains three integers nnn, qqq and kkk (1≤n,q≤1051≤n,q≤10^51≤n,q≤105, n≤k≤109n≤k≤10^9n≤k≤109) — the length of array aaa, the number of queries and number kkk.

The second line contains nnn integers a1,a2,…,ana_1,a_2,…,a_na1​,a2​,…,an​ (1≤ai≤k1≤a_i≤k1≤ai​≤k). This array is strictly increasing — a1<a2<…<ana_1<a_2<…<a_na1​<a2​<…<an​.

Each of the following qqq lines contains two integers lil_ili​, rir_iri​ (1≤li≤ri≤n1≤l_i≤r_i≤n1≤li​≤ri​≤n).

Output

Print qqq lines. The iii-th of them should contain the answer to the iii-th query.
