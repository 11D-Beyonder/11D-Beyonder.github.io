HDU 2243 考研路茫茫——单词情结

发表于2021-04-12|更新于2023-01-31|题解
|字数总计:1.8k|阅读时长:9分钟|阅读量:

Problem Description

背单词，始终是复习英语的重要环节。在荒废了3年大学生涯后，Lele 也终于要开始背单词了。
一天，Lele 在某本单词书上看到了一个根据词根来背单词的方法。比如 “ab”“ab”“ab”,放在单词前一般表示“相反，变坏，离去"等。

于是Lele想，如果背了 nnn 个词根，那这些词根到底会不会在单词里出现呢。更确切的描述是：长度不超过 lll，只由小写字母组成的，至少包含一个词根的单词，一共可能有多少个呢？这里就不考虑单词是否有实际意义。

比如一共有 222 个词根 “aa”“aa”“aa” 和 “ab”“ab”“ab” ，则可能存在 104104104 个长度不超过 333 的单词，分别为

  • (2个) aa,abaa,abaa,ab
  • (26个)aaa,aab,aac⋯aazaaa,aab,aac\cdots aazaaa,aab,aac⋯aaz
  • (26个)aba,abb,abc⋯abzaba,abb,abc\cdots abzaba,abb,abc⋯abz
  • (25个)baa,caa,daa⋯zaabaa,caa,daa\cdots zaabaa,caa,daa⋯zaa
  • (25个)bab,cab,dab⋯zabbab,cab,dab\cdots zabbab,cab,dab⋯zab

这个只是很小的情况。而对于其他复杂点的情况，Lele 实在是数不出来了，现在就请你帮帮他。

Input

本题目包含多组数据，请处理到文件结束。
每组数据占两行。
第一行有两个正整数 nnn 和 lll（0<n<60<n<60<n<6, 0<l<2310<l<2^{31}0<l<231）。
第二行有 nnn 个词根，每个词根仅由小写字母组成，长度不超过 555。两个词根中间用一个空格分隔开。

Output

对于每组数据，请在一行里输出一共可能的单词数目。

由于结果可能非常巨大，你只需要输出单词总数模 2642^{64}264 的值。

Idea

我们可以依据容斥原理，长度不超过 lll 的所有小写字母组成的串有 ∑i=1l26i\sum\limits_{i=1}^l 26^ii=1∑l​26i 个；若能求出不包含任何一个词根的串的个数 ttt，则答案为 ∑i=1l26i−t\sum\limits_{i=1}^l26^i-ti=1∑l​26i−t。

类似 POJ 2778 DNA Sequence，我们构建AC自动机；自动机上将一切词根禁止，那么在自动机上游走 kkk（1≤k≤l1\le k\le l1≤k≤l）步的方案数就是不包含任何词根的串的个数 ttt。自动机的邻接矩阵为 AAA，在AC自动机上游走 kkk 步，∑i=0totA0,ik\sum\limits_{i=0}^{tot}A_{0,i}^ki=0∑tot​A0,ik​ 即为答案，tottottot 为AC自动机上最大点的编号，树根编号为 000。要构造长度不超过 lll 的串，总方案数 t=∑k=1l∑i=0totA0,ik=∑i=0tot(∑k=1lAk)0,it=\sum\limits_{k=1}^l\sum\limits_{i=0}^{tot}A_{0,i}^k=\sum\limits_{i=0}^{tot}\left(\sum\limits_{k=1}^lA^k\right)_{0,i}t=k=1∑l​i=0∑tot​A0,ik​=i=0∑tot​(k=1∑l​Ak)0,i​。

由于数据量大，答案需要模 2642^{64}264，可以直接用 unsigned long long\text{unsigned long long}unsigned long long 自然溢出。对于等比数列求和 ∑i=1l26i\sum\limits_{i=1}^{l}26^ii=1∑l​26i，直接利用求和公式需要求 252525 模 2642^{64}264 意义下的逆元，好麻烦，还得扩展欧几里得；对于等比矩阵求和 ∑k=1lAk\sum\limits_{k=1}^l A^kk=1∑l​Ak，利用求和公式还需要求逆矩阵，麻烦死了，还要高斯消元，增加复杂度。因此不妨直接递归求等比数列、矩阵的和。
