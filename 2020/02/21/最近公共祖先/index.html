最近公共祖先

发表于2020-02-21|更新于2023-01-31|算法
|字数总计:1.3k|阅读时长:6分钟|阅读量:

定义

在图论和计算机科学中，最近公共祖先是指在一颗树或者一个有向无环图中同时拥有vvv和www作为后代的最深的节点。在这里，我们定义一个节点也是其自己的后代，因此如果vvv是www的后代，那么www就是vvv和www的最近公共祖先。(来源:维基百科)

每两个点的最近公共祖先一目了然。

用树上倍增求两点的LCA

思想

如图，是一棵树

每两个节点的最近公共祖先一目了然。
假设我们要寻找x,yx,yx,y的最近公共祖先（默认xxx的深度比yyy的深度大），只需要做到如下步骤：
①利用倍增算法，将xxx上升至同yyy的相同深度；
②同步上升x,yx,yx,y，直到两节点的父亲节点相同；
③返回此时相同的父亲节点。
father[n][logn]father[n][logn]father[n][logn]和每一个节点的深度depth[n]depth[n]depth[n]可以通过BFS\text{BFS}BFS得到，上升的步骤则利用倍增算法。

用LCA求树上两点的距离

思想

如图，是一棵树

设两点的编号为u,vu,vu,v(默认uuu的深度比vvv的深度大),两点的位置关系会有如下情况。
定义dis[i]dis[i]dis[i]为第iii个节点到根节点的距离。
①u,vu,vu,v在一条链上,那么距离为dis[u]−dis[v]dis[u]-dis[v]dis[u]−dis[v];
②u,vu,vu,v不在一条链上,那么我们必须要先从uuu出发,走到u,vu,vu,v的最近公共祖先,再走到vvv,此时距离 为dis[u]−dis[lca(u,v)]+dis[v]−dis[lca(u,v)]=dis[u]+dis[v]−2×dis[lca(u,v)]dis[u]-dis[lca(u,v)]+dis[v]-dis[lca(u,v)]=dis[u]+dis[v]-2×dis[lca(u,v)]dis[u]−dis[lca(u,v)]+dis[v]−dis[lca(u,v)]=dis[u]+dis[v]−2×dis[lca(u,v)]。
综上所述,u,vu,vu,v的距离始终为dis[u]+dis[v]−2×dis[lca(u,v)]dis[u]+dis[v]-2×dis[lca(u,v)]dis[u]+dis[v]−2×dis[lca(u,v)]

模板 HDU 2856 How far away ？

Problem Description

There are nnn houses in the village and some bidirectional roads connecting them. Every day peole always like to ask like this “How far is it if I want to go from house AAA to house BBB”? Usually it hard to answer. But luckily in this village the answer is always unique, since the roads are built in the way that there is a unique simple path(“simple” means you can’t visit a place twice) between every two houses. Yout task is to answer all these curious people.

Input

First line is a single integer TTT(((T<=10T<=10T<=10))), indicating the number of test cases.
For each test case,in the first line there are two numbers nnn(2≤n≤40000)(2\leq n\leq 40000)(2≤n≤40000) and mmm (1≤m≤200)(1\leq m\leq 200)(1≤m≤200),the number of houses and the number of queries. The following n−1n-1n−1 lines each consisting three numbers i,j,ki,j,ki,j,k, separated by a single space, meaning that there is a road connecting house iii and house jjj,with length kkk(0<k≤40000)(0<k\leq 40000)(0<k≤40000).The houses are labeled from 111 to nnn.
Next mmm lines each has distinct integers iii and jjj, you are supposed to answer the distance between house iii and house jjj.

Output

For each test case,output mmm lines. Each line represents the answer of the query. Output a bland line after each test case.

Sample Input

2
3 2
1 2 10
3 1 15
1 2
2 3
2 2
1 2 100
1 2
2 1

Sample Output

10
25
100
100

Translation

给定一棵 nnn 个点的树，mmm 次询问，每次询问输出 x,yx,yx,y 两点之间的最短距离。
