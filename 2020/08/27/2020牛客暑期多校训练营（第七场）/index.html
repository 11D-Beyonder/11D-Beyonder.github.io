“I learn math since I was born and I am very good at it,”, said Mr. Thumb," People keep asking me why I know math so much and I sometimes find myself having those amazing ideas about math."

H. Dividing ↬\looparrowright↬

题目描述

The following rules define a kind of integer tuple - the Legend Tuple:
(1,k)(1, k)(1,k) is always a Legend Tuple, where k is an integer.
if (n,k)(n, k)(n,k) is a Legend Tuple, (n+k,k)(n + k, k)(n+k,k) is also a Legend Tuple.
if (n,k)(n, k)(n,k) is a Legend Tuple, (nk,k)(nk, k)(nk,k) is also a Legend Tuple.
We want to know the number of the Legend Tuples (n,k)(n, k)(n,k) where 1⩽n⩽N1 \leqslant n \leqslant N1⩽n⩽N, 1⩽k⩽K1 \leqslant k \leqslant K1⩽k⩽K.
In order to avoid calculations of huge integers, report the answer modulo 109+710^9+7109+7 instead.

输入描述

The input contains two integers NNN and KKK, 1⩽N,K⩽10121 \leqslant N, K \leqslant 10^{12}1⩽N,K⩽1012.

输出描述

Output the answer modulo 109+710^9+7109+7.

所有 Legend Tuple 从 (1,k)(1,k)(1,k) 出发，会产生 (xk,k)(xk,k)(xk,k) 和 (xk+1,k)(xk+1,k)(xk+1,k) 两种情况（x∈N∗x\in\mathbb N^\astx∈N∗）；其中，(xk+1,k)(xk+1,k)(xk+1,k) 的情况经过一步操作可以变成 (xk,k)(xk,k)(xk,k) 的情况。
显然，当且仅当 k∣(n−1)k\mid (n-1)k∣(n−1) 或 k∣nk\mid nk∣n 时，(n,k)(n,k)(n,k) 为 Legend Tuple。
对于 k∣(n−1)k\mid(n-1)k∣(n−1) 的情况，其贡献为 ∑k=1K∑n=1N[k∣(n−1)]=∑k=1K⌊N−1k⌋+K\sum\limits_{k=1}^K\sum\limits_{n=1}^N\left[k\mid(n-1)\right]=\sum\limits_{k=1}^K\left\lfloor\frac{N-1}{k}\right\rfloor+Kk=1∑K​n=1∑N​[k∣(n−1)]=k=1∑K​⌊kN−1​⌋+K；∑k=1K⌊N−1k⌋\sum\limits_{k=1}^K\left\lfloor\frac{N-1}{k}\right\rfloork=1∑K​⌊kN−1​⌋ 为 n⩾2n\geqslant 2n⩾2 的贡献，KKK 为 n=1n=1n=1 的贡献。
对于 k∣nk\mid nk∣n 的情况，其贡献为 ∑k=1K∑n=1N[k∣n]=∑k=1K⌊Nk⌋\sum\limits_{k=1}^K\sum\limits_{n=1}^N [k\mid n]=\sum\limits_{k=1}^K\left\lfloor\frac{N}{k}\right\rfloork=1∑K​n=1∑N​[k∣n]=k=1∑K​⌊kN​⌋。
对于两部分贡献，有 (1,1),(2,1),⋯ ,(N,1)(1,1),(2,1),\cdots,(N,1)(1,1),(2,1),⋯,(N,1) 这部分的贡献是重复的，需要减去重复的一份贡献 NNN。
因此最终的答案为 ∑k=1K∑n=1N[k∣n]=∑k=1K⌊Nk⌋+∑k=1K∑n=1N[k∣(n−1)]=∑k=1K⌊N−1k⌋+K−N\sum\limits_{k=1}^K\sum\limits_{n=1}^N [k\mid n]=\sum\limits_{k=1}^K\left\lfloor\frac{N}{k}\right\rfloor+\sum\limits_{k=1}^K\sum\limits_{n=1}^N\left[k\mid(n-1)\right]=\sum\limits_{k=1}^K\left\lfloor\frac{N-1}{k}\right\rfloor+K-Nk=1∑K​n=1∑N​[k∣n]=k=1∑K​⌊kN​⌋+k=1∑K​n=1∑N​[k∣(n−1)]=k=1∑K​⌊kN−1​⌋+K−N。可以用数论分块解决。
当然，N=1N=1N=1 或 K=1K=1K=1 时需要进行特判：当 N=1N=1N=1，答案为 KKK；当 K=1K=1K=1，答案为 NNN。
