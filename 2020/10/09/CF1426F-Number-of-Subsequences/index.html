CF1426F Number of Subsequences

发表于2020-10-09|更新于2023-01-31|题解
|字数总计:1.5k|阅读时长:8分钟|阅读量:

You are given a string s consisting of lowercase Latin letters “a”, “b” and “c” and question marks “?”.
Let the number of question marks in the string sss be kkk. Let’s replace each question mark with one of the letters “a”, “b” and “c”. Here we can obtain all 3k3^k3k possible strings consisting only of letters “a”, “b” and “c”. For example, if s=“ac?b?c” then we can obtain the following strings: [“acabac”, “acabbc”, “acabcc”, “acbbac”, “acbbbc”, “acbbcc”, “accbac”, “accbbc”, “accbcc”].
Your task is to count the total number of subsequences “abc” in all resulting strings. Since the answer can be very large, print it modulo 109+710^9+7109+7.
A subsequence of the string ttt is such a sequence that can be derived from the string ttt after removing some (possibly, zero) number of letters without changing the order of remaining letters. For example, the string “baacbc” contains two subsequences “abc” — a subsequence consisting of letters at positions (2,5,6)(2,5,6)(2,5,6) and a subsequence consisting of letters at positions (3,5,6)(3,5,6)(3,5,6).

Input

The first line of the input contains one integer nnn (3≤n≤200000)(3≤n≤200000)(3≤n≤200000) — the length of sss.
The second line of the input contains the string sss of length nnn consisting of lowercase Latin letters “a”, “b” and “c” and question marks"?".

Output

Print the total number of subsequences “abc” in all strings you can obtain if you replace all question marks with letters “a”, “b” and “c”, modulo 109+710^9+7109+7.

Note

In the first example, we can obtain 999 strings:

  • “acabac” — there are 222 subsequences “abc”,
  • “acabbc” — there are 444 subsequences “abc”,
  • “acabcc” — there are 444 subsequences “abc”,
  • “acbbac” — there are 222 subsequences “abc”,
  • “acbbbc” — there are 333 subsequences “abc”,
  • “acbbcc” — there are 444 subsequences “abc”,
  • “accbac” — there is 111 subsequence “abc”,
  • “accbbc” — there are 222 subsequences “abc”,
  • “accbcc” — there are 222 subsequences “abc”.

So, there are 2+4+4+2+3+4+1+2+2=242+4+4+2+3+4+1+2+2=242+4+4+2+3+4+1+2+2=24 subsequences “abc” in total.
