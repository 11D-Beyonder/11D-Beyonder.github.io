A - The Number of Even Pairs

Time Limit: 2 sec / Memory Limit: 1024 MB
Score : 100100100 points

Problem Statement

We have N+MN+MN+M balls, each of which has an integer written on it.
It is known that:
The numbers written on NNN of the balls are even.
The numbers written on MMM of the balls are odd.
Find the number of ways to choose two of the N+MN+MN+M balls (disregarding order) so that the sum of the numbers written on them is even.
It can be shown that this count does not depend on the actual values written on the balls.

Constraints

0≤N,M≤1000≤N,M≤1000≤N,M≤100
2≤N+M2≤N+M2≤N+M
All values in input are integers.

Input

Input is given from Standard Input in the following format:
NNN MMM

Output

Print the answer.

Sample Input 1

2 1

Sample Output 1

1
For example, let us assume that the numbers written on the three balls are 1,2,41,2,41,2,4.
If we choose the two balls with 111and 222, the sum is odd;
If we choose the two balls with 111 and 444, the sum is odd;
If we choose the two balls with 222 and 444, the sum is even.
Thus, the answer is 111.

Sample Input 2

4 3

Sample Output 2

9

Sample Input 3

1 1

Sample Output 3

0

Sample Input 4

13 3

Sample Output 4

81

Sample Input 5

0 3

Sample Output 5

3

Translation

有N+MN+MN+M个球，NNN个球上面写着偶数，MMM个球上面写着奇数，要从这N+MN+MN+M个球中选取两个球，使得两个球上写的数字的和为偶数，问有多少种选法。

Idea

同奇偶性的两个数相加为偶数，所以答案为Cn2+Cm2C_n^2+C_m^2Cn2​+Cm2​。

B - String Palindrome

Time Limit: 2 sec / Memory Limit: 1024 MB
Score : 200200200 points

Problem Statement

A string SSS of an odd length is said to be a strong palindrome if and only if all of the following conditions are satisfied:
SSS is a palindrome.
Let NNN be the length of SSS. The string formed by the 111-st through ((N−1)/2)((N−1)/2)((N−1)/2)-th characters of SSS is a palindrome.
The string consisting of the (N+3)/2(N+3)/2(N+3)/2-st through NNN-th characters of SSS is a palindrome.
Determine whether SSS is a strong palindrome.

Constraints

SSS consists of lowercase English letters.
The length of SSS is an odd number between 333 and 999999 (inclusive).

Input

Input is given from Standard Input in the following format:
SSS

Output

If SSS is a strong palindrome, print Yes; otherwise, print No.

Sample Input 1

akasaka

Sample Output 1

Yes
SSS is akasaka.
The string formed by the 111-st through the 333-rd characters is aka.
The string formed by the 555-th through the 777-th characters is aka. All of these are palindromes, so SSS is a strong palindrome.

Sample Input 2

level

Sample Output 2

No

Sample Input 3

atcoder

Sample Output 3

No

Translation

给一个字符串SSS，要求判断该字符串是不是strong palindrome\text {strong palindrome}strong palindrome（条件见英文题面）。

Idea

按题给意思模拟。
用reverse\text {reverse}reverse函数判断是否回文。

C - Maximum Volume

Time Limit: 2 sec / Memory Limit: 1024 MB
Score : 300300300 points

Problem Statement

Given is a positive integer LLL. Find the maximum possible volume of a rectangular cuboid whose sum of the dimensions (not necessarily integers) is LLL.

Constraints

1≤L≤10001≤L≤10001≤L≤1000
LLL is an integer.

Input

Input is given from Standard Input in the following format:
LLL

Output

Print the maximum possible volume of a rectangular cuboid whose sum of the dimensions (not necessarily integers) is LLL. Your output is considered correct if its absolute or relative error from our answer is at most 10−610^{-6}10−6.

Sample Input 1

3

Sample Output 1

1.000000000000
For example, a rectangular cuboid whose dimensions are 0.80.80.8, 111, and 1.21.21.2 has a volume of 0.960.960.96.
On the other hand, if the dimensions are 111, 111, and 111, the volume of the rectangular cuboid is 111, which is greater.

Sample Input 2

999

Sample Output 2

36926037.000000000000

Translation

长方体的长、宽、高之和为LLL，问其最大体积。

Idea

由AM-GM\text{AM-GM}AM-GM不等式，$\sqrt[n]{\prod\limits_{i=1}^{n}x_i}\leqslant \frac{\sum\limits_{i=1}^{n}x_i}{n}$，当且仅当x1=x2=⋯=xnx_1=x_2=\dots=x_nx1​=x2​=⋯=xn​时取等号。
故$V_{max}=\frac{L}{3}\times\frac{L}{3}\times\frac{L}{3}=\frac{L^3}{27}$。

E - Dividing Chocolate

Time Limit: 2 sec / Memory Limit: 1024 MB
Score : 500500500 points

Problem Statement

We have a chocolate bar partitioned into HHH horizontal rows and WWW vertical columns of squares.
The square (i,j)(i,j)(i,j) at the i-th row from the top and the jjj-th column from the left is dark if Si,jS_{i,j}Si,j​ is 000 and white if Si,jS_{i,j}Si,j​ is 111.
We will cut the bar some number of times to divide it into some number of blocks. In each cut, we cut the whole bar by a line running along some boundaries of squares from end to end of the bar.
How many times do we need to cut the bar so that every block after the cuts has KKK or less white squares?

Constraints

1≤H≤101≤H≤101≤H≤10
1≤W≤10001≤W≤10001≤W≤1000
1≤K≤H×W1≤K≤H×W1≤K≤H×W
Si,jS_{i,j}Si,j​ is 000 or 111.

Input

Input is given from Standard Input in the following format:
HHH WWW KKK
S1,1S1,2...S1,WS_{1,1}S_{1,2}...S_{1,W}S1,1​S1,2​...S1,W​
:::
SH,1SH,2...SH,WS_{H,1}S_{H,2}...S_{H,W}SH,1​SH,2​...SH,W​

Output

Print the number of minimum times the bar needs to be cut so that every block after the cuts has KKK or less white squares.

Sample Input 1

3 5 4
11100
10001
00111

Sample Output 1

2
For example, cutting between the 111-st and 222-nd rows and between the 333-rd and 444-th columns - as shown in the figure to the left - works.
Note that we cannot cut the bar in the ways shown in the two figures to the right.

Sample Input 2

3 5 8
11100
10001
00111

Sample Output 2

0
No cut is needed.

Sample Input 3

4 10 4
1110010010
1000101110
0011101001
1101000111

Sample Output 3

3

Translation

给定一个n×mn×mn×m的01字符矩阵。求最少需要多少次水平或竖直切割，使得切出的每一块都包含不超过sss个1。

Idea

nnn的范围很小，可以枚举2n2^{n}2n种水平切割情况，竖直切割情况则用贪心策略检验。
对于每一种水平切割的方案，按列从左往右贪心，每到一列，水平切割切成的每一块当前的1的个数就会增加该列1的个数，如果至少有一块当前的1的个数超过了sss，便在本列与上一列之间竖直切割一刀，如果仍然超过，那么此种水平切割方案就不可行。对于每种水平切割状态，如果可行就需要更新答案。
时间复杂度O(2nnm)O(2^nnm)O(2nnm)。

F - Knapsack for All Segments

Time Limit: 2 sec / Memory Limit: 1024 MB
Score : 600600600 points

Problem Statement

Given are a sequence of NNN integers A1,A2,…,ANA_1, A_2, …, A_NA1​,A2​,…,AN​ and a positive integer SSS.
For a pair of integers (L,R)(L,R)(L,R) such that 1≤L≤R≤N1≤L≤R≤N1≤L≤R≤N, let us define f(L,R)f(L,R)f(L,R) as follows:
f(L,R)f(L,R)f(L,R) is the number of sequences of integers (x1,x2,…,xk)(x_1,x_2,…,x_k)(x1​,x2​,…,xk​) such that L≤x1<x2<⋯<xk≤RL≤x_1<x_2<⋯<x_k≤RL≤x1​<x2​<⋯<xk​≤R and Ax1+Ax2+⋯+Axk=SA_{x_1}+A_{x_2}+⋯+A_{x_k}=SAx1​​+Ax2​​+⋯+Axk​​=S.
Find the sum of f(L,R)f(L,R)f(L,R) over all pairs of integers (L,R)(L,R)(L,R) such that 1≤L≤R≤N1≤L≤R≤N1≤L≤R≤N. Since this sum can be enormous, print it modulo 998244353998244353998244353.

Constraints

All values in input are integers.
1≤N≤30001≤N≤30001≤N≤3000
1≤S≤30001≤S≤30001≤S≤3000
1≤Ai≤30001≤A_i≤30001≤Ai​≤3000

Input

Input is given from Standard Input in the following format:
NNN SSS
A1A2...ANA_1 A_2 ...A_NA1​A2​...AN​

Output

Print the sum of f(L,R)f(L,R)f(L,R), modulo 998244353998244353998244353.

Sample Input 1

3 4
2 2 4

Sample Output 1

5
The value of f(L,R)f(L,R)f(L,R) for each pair is as follows, for a total of 555.
f(1,1)=0f(1,1)=0f(1,1)=0
f(1,2)=1f(1,2)=1f(1,2)=1 (for the sequence (1,2)(1,2)(1,2))
f(1,3)=2f(1,3)=2f(1,3)=2 (for (1,2)(1,2)(1,2) and (3)(3)(3))
f(2,2)=0f(2,2)=0f(2,2)=0
f(2,3)=1f(2,3)=1f(2,3)=1 (for (3)(3)(3))
f(3,3)=1f(3,3)=1f(3,3)=1 (for (3)(3)(3))

Sample Input 2

5 8
9 9 9 9 9

Sample Input 3

3 1 4 1 5 9 2 6 5 3

Translation

给定一个长度为 NNN 的数组 AAA 和一个正整数 SSS ，定义 f(L,R)f(L,R)f(L,R) 为原数组中满足：L≤x1≤x2≤⋯≤xk≤RL\leq x_1\leq x_2\leq\cdots\leq x_k \leq RL≤x1​≤x2​≤⋯≤xk​≤R 并且 Ax1+Ax2+⋯+Axk=SA_{x_1}+A_{x_2}+\cdots + A_{x_k}=SAx1​​+Ax2​​+⋯+Axk​​=S 的子序列数量。求 ∑L=1N∑R=LNf(L,R)\sum\limits^N_{L=1}\sum\limits^N_{R=L}f(L,R)L=1∑N​R=L∑N​f(L,R) 。

Idea

dp[j]dp[j]dp[j]表示当前和为 jjj 的子序列数量，那么本题的方案数就可以通过 010101 背包的方法转移。考虑新加入一个数字 AiA_iAi​ 时的转移，首先对于新加入的元素，我们只需要考虑新加入元素与先前元素构成的新子序列，即只考虑 f(L,R)f(L,R)f(L,R) 右端点变化，这一过程通过 010101 背包转移方案数；然后， f(L,R)f(L,R)f(L,R) 左端点的选择是在 [1,i][1,i][1,i] 范围内任意的，因此加入的 AiA_iAi​ 对后继的影响是： dp[A[i]]=dp[A[i]]+idp\left[A\left[i\right]\right]=dp\left[A\left[i\right]\right]+idp[A[i]]=dp[A[i]]+i 。
