HDU 1495 非常可乐

发表于2020-03-10|更新于2023-01-31|题解
|字数总计:1.4k|阅读时长:6分钟|阅读量:

Problem Description

大家一定觉的运动以后喝可乐是一件很惬意的事情，但是seeyou却不这么认为。因为每次当seeyou买了可乐以后，阿牛就要求和seeyou一起分享这一瓶可乐，而且一定要喝的和seeyou一样多。但seeyou的手中只有两个杯子，它们的容量分别是NNN 毫升和MMM 毫升 可乐的体积为S（S<101）S （S<101）S（S<101）毫升　(正好装满一瓶) ，它们三个之间可以相互倒可乐 (都是没有刻度的，且 S=N+M，101＞S＞0，N＞0，M＞0S=N+M，101＞S＞0，N＞0，M＞0S=N+M，101＞S＞0，N＞0，M＞0) 。聪明的ACMER你们说他们能平分吗？如果能请输出倒可乐的最少的次数，如果不能输出"NO"。

Input

三个整数 : SSS 可乐的体积 , NNN 和 MMM是两个杯子的容量，以"0 0 0"结束。

Output

如果能平分的话请输出最少要倒的次数，否则输出"NO"。

Sample Input

7 4 3
4 1 3
0 0 0

Sample Output

NO
3

Idea1

每一次操作有选取两个瓶子，并从中选取一个瓶子倒入另一个瓶子，因此每一次操作有C32×C21=6C_3^2 \times C_2^1=6C32​×C21​=6种选择。
于是可以利用BFSBFSBFS枚举每一次状态的六种选择。

Idea2

设两个小瓶子容积分别为a,ba,ba,b，问题转化成通过两个小瓶子的若干次倒进或倒出操作得到(a+b)/2体积的可乐，设两个小瓶子被倒进或倒出xxx次和yyy次；这里的xxx和yyy是累加后的操作即x=第一个瓶子倒出的次数−倒进的次数，y=第二个瓶子倒出的次数−倒进的次数x=第一个瓶子倒出的次数-倒进的次数，y=第二个瓶子倒出的次数-倒进的次数x=第一个瓶子倒出的次数−倒进的次数，y=第二个瓶子倒出的次数−倒进的次数。
那么问题转化成：

所以∣x∣+∣y∣|x|+|y|∣x∣+∣y∣的最小值为c+d2=s2gcd(a,b)\frac {c+d}{2}=\frac{s}{2gcd(a,b)}2c+d​=2gcd(a,b)s​，通过xxx和yyy的通解形式显然可以看出xxx和yyy一正一负，不妨设x<0x<0x<0，那么就是往第一个小瓶子倒进xxx次，第二个小瓶子倒出yyy次，但是由于瓶子容积有限，所以倒进倒出操作都是通过大瓶子来解决的，一次倒进操作后为了继续使用小瓶子还要将小瓶子中可乐倒回大瓶子中，倒出操作同理，所以总操作次数是c+d2×2=c+d=sgcd(a,b)\frac{c+d}{2}\times 2=c+d=\frac {s}{gcd(a,b)}2c+d​×2=c+d=gcd(a,b)s​，但是注意最后剩下的s2\frac{s}{2}2s​体积的可乐一定是放在两个小瓶子中较大的那个中，而不是再倒回到大瓶子中，所以操作数要减一，答案就是sgcd(a,b)−1\frac {s}{gcd(a,b)}-1gcd(a,b)s​−1。
