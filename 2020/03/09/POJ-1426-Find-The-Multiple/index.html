POJ 1426 Find The Multiple

发表于2020-03-09|更新于2023-01-31|题解
|字数总计:777|阅读时长:4分钟|阅读量:

Description

Given a positive integer nnn, write a program to find out a nonzero multiple m of nnn whose decimal representation contains only the digits 000 and 111. You may assume that nnn is not greater than 200 and there is a corresponding m containing no more than 100100100 decimal digits.

Input

The input file may contain multiple test cases. Each line contains a value of n(1<=n<=200)n (1 <= n <= 200)n(1<=n<=200). A line containing a zero terminates the input.

Output

For each value of nnn in the input print a line containing the corresponding value of mmm. The decimal representation of mmm must not contain more than 100100100 digits. If there are multiple solutions for a given value of nnn, any one of them is acceptable.

Sample Input

2
6
19
0

Sample Output

10
100100100100100100
111111111111111111

Translation

输入一个不大于200200200的正整数nnn，输出一个nnn的倍数，且这个倍数只能由000和111构成。

Idea

逐项检验肯定是不行的，我们不妨想想怎么构造010101串。
要使构造的010101串xxx成为nnn的倍数，那么最小的xxx就是111，之后就是10,11,100,101,110,111...10,11,100,101,110,111...10,11,100,101,110,111...
可见，对于任何一个010101串xxx，都可延伸出两个010101串10x+1,10x10x+1,10x10x+1,10x，就这样从111开始，形成了一颗无限的完全二叉树，这样的数据结构，适合BFSBFSBFS构造。
于是可以枚举[1,200][1,200][1,200]之间的全部整数，找到它们符合条件的倍数。
