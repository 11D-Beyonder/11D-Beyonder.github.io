POJ 3087 Shuffle'm Up

发表于2020-03-13|更新于2023-01-31|题解
|字数总计:813|阅读时长:4分钟|阅读量:

Description

A common pastime for poker players at a poker table is to shuffle stacks of chips. Shuffling chips is performed by starting with two stacks of poker chips, S1S_1S1​ and S2S_2S2​, each stack containing CCC chips. Each stack may contain chips of several different colors.
The actual shuffle operation is performed by interleaving a chip from S1S_1S1​ with a chip from S2S_2S2​ as shown below for C=5C=5C=5:

The single resultant stack, S12S_{12}S12​, contains 2C2C2C chips. The bottommost chip of S12S_{12}S12​ is the bottommost chip from S2S_2S2​. On top of that chip, is the bottommost chip from S1S_1S1​. The interleaving process continues taking the 2nd2^{nd}2nd chip from the bottom of S2S_2S2​ and placing that on S12S_{12}S12​, followed by the 2nd chip from the bottom of S1S_1S1​ and so on until the topmost chip from S1S_1S1​ is placed on top of S12S_{12}S12​.
After the shuffle operation, S12S_{12}S12​ is split into 222 new stacks by taking the bottommost CCC chips from S12S_{12}S12​ to form a new S1S_1S1​ and the topmost CCC chips from S12S_{12}S12​ to form a new S2S_2S2​. The shuffle operation may then be repeated to form a new S12S_{12}S12​.
For this problem, you will write a program to determine if a particular resultant stack S12S_{12}S12​ can be formed by shuffling two stacks some number of times.

Input

The first line of input contains a single integer N(1≤N≤1000)N (1 ≤ N ≤ 1000)N(1≤N≤1000) which is the number of datasets that follow.
Each dataset consists of four lines of input. The first line of a dataset specifies an integer CCC, (1≤C≤100)(1 ≤ C ≤ 100)(1≤C≤100) which is the number of chips in each initial stack (S1S_1S1​ and S2S_2S2​). The second line of each dataset specifies the colors of each of the C chips in stack S1S_1S1​, starting with the bottommost chip. The third line of each dataset specifies the colors of each of the C chips in stack S2S_2S2​ starting with the bottommost chip. Colors are expressed as a single uppercase letter (A through H). There are no blanks or separators between the chip colors. The fourth line of each dataset contains 2C2C2C uppercase letters (A through H), representing the colors of the desired result of the shuffling of S1S_1S1​ and S2S_2S2​ zero or more times. The bottommost chip’s color is specified first.

Output

Output for each dataset consists of a single line that displays the dataset number (111 though NNN), a space, and an integer value which is the minimum number of shuffle operations required to get the desired resultant stack. If the desired result can not be reached using the input for the dataset, display the value negative 1(−1)1 (−1)1(−1) for the number of shuffle operations.

Sample Input

2
4
AHAH
HAHA
HHAAAAHH

CDE
CDE
EEDDCC

Sample Output

1 2
2 -1

Translation

给定两个长度为lenlenlen的字符串S1S_1S1​和S2S_2S2​， 接着给出一个长度为len×2len\times 2len×2的字符串S12S_{12}S12​。
将字符串S1S_1S1​和S2S_2S2​通过一定的变换变成S12S_{12}S12​，找到最小变换次数。
变换规则如下：
假设S1=AHAH,S2=HAHAS_1=AHAH,S_2=HAHAS1​=AHAH,S2​=HAHA，变换后的序列 S12=HAAHHAAHS_{12}=HAAHHAAHS12​=HAAHHAAH。类似于洗牌的过程。

Idea

直接暴力模拟洗牌过程，得到目标态就输出变换次数；判断不可能的条件是某一状态前面已经出现过，这样就出现了死循环（用set标记）。
