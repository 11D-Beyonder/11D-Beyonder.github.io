PennyLane的使用

发表于2022-07-07|更新于2023-01-31|课程笔记
|字数总计:667|阅读时长:2分钟|阅读量:

PennyLane是跨平台的量子机器学习Python包。

量子电路

量子函数

一个量子电路可以视为一个Python函数，称为量子函数。

PennyLane使用wire指代一个量子系统，在大多数设备上就代表一个量子比特。

量子函数遵守以下规则：

  • 量子函数接受经典的输入，由量子运算符或称为模板的运算符序列组成，每行使用一条指令；
  • 函数可以包含经典的控制结构，如 for 循环或 if 语句；
  • 函数必须以单个值或元组的方式返回量子的观测值。

定义设备

要运行一个量子电路，首先需要指定一个计算设备。设备是 Device 类的实例，可以表示模拟器或硬件设备。

PennyLane支持的设备见Plugins and ecosystem — PennyLane。

设备的选择显著地决定了计算速度，以及可以传递给设备加载程序的可用选项。

wire参数可以是一个整数，定义了系统中wire的数量，可以用连续整数（0、1、2……）索引。你也可以自定义wire的标签，在量子函数中可以使用自定义的标签来索引wire。

shot表示为估计统计量对电路进行采样的次数。在一些受支持的模拟器设备上，shot = None精确地计算统计信息。你还可以传入一个整数列表，表示多个采样批次，返回多组结果。

量子节点

量子函数和设备一起用于创建量子节点或 QNode 对象，该对象包装量子函数并将其绑定到设备。

QNode 可以用来计算量子电路的结果，就好像它是一个标准的 Python 函数一样。

tensor(0.748, requires_grad=True)
